## Setup 🚀

1. Clone the repository
```bash
git clone https://github.com/AjmalDevala/my-portfolio-react.git
cd my-portfolio-react
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Visit `http://localhost:3000`
